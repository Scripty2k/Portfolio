A simple thing with this function is that you can specify something by something else

For example, like this:
```python
monthconvert = {  
    'jan': 'January',  
    'feb': 'February',  
    'mar': 'March',  
    'apr': 'April',  
    'May': 'May',  
    'June': 'June',  
    'July': 'July',  
    'aug': 'August',  
    'sept': 'September',  
    'oct': 'October',  
    'nov': 'November',  
    'dec': 'December'  
}  
  
print(monthconvert['sept'])
```
